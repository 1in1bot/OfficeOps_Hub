핵심 사용자 시나리오

  1. 1직원이 요청 유형을 선택하고 상세 내용을 입력해 요청을 등록한다.
  2. 2팀장이 소속 팀원의 요청을 확인하고 1차 승인 또는 반려한다.
  3. 3운영 담당자가 최종 승인하며 담당자, 처리 예정일, 마감일을 지정한다.
  4. 4담당자가 처리 중으로 변경한 뒤 완료 처리하고, 모든 상태 변경은 이력으로 남긴다.
