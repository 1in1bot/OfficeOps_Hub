v1.1.0 삭제: `/signup` 공개 회원가입 카드와 로그인-회원가입 연결 제거

v1.2.0 수정: 문서 파일명과 버전 표기를 새 규칙으로 정리

v1.3.0 수정: 카드 크기, 색상, 간격, 직선 화살표, 선택 강조 방식 조정

v1.4.0 수정: 버전 이력 접기 UI와 마우스 드래그 화면 이동 추가

v1.4.1 수정: 카드 클릭 강조 유지, 빈 영역 드래그 이동, 사용자 계정 생성 카드 배치 조정

v1.5.0 수정: 연결된 모든 카드 강조, 배경 클릭 해제, 카드 헤더 드래그 이동과 자동 겹침 보정 추가

v1.5.1 수정: 배경 클릭 강조 해제 보강, 카드 이동 중 기존 카드 위치 고정, 드래그 종료 시 이동 카드만 빈 위치로 정렬

v1.5.2 수정: 드래그 종료 시 전체 카드가 간격과 줄 맞춤을 유지하며 자동 재배치되도록 정렬 애니메이션 추가

v1.5.3 수정: 충돌이 없을 때 자유 배치를 유지하고, 충돌 시 주변 카드 행/열 기준으로 자연스럽게 정렬되도록 보완

v1.5.4 수정: 기본 카드 배치를 정돈하고, 충돌 시 기존 카드가 과도하게 튀지 않게 같은 줄 방향으로 자연스럽게 밀리도록 보완

v1.5.5 수정: 근처 카드의 상하좌우 슬롯에 자석처럼 붙고, 같은 줄 카드만 최소 범위로 밀리도록 배치 흐름 보완

v1.5.6 수정: 기본 카드 배치를 최소 간격 기준으로 재정렬하고, 카드 위 배치 대신 같은 줄 카드가 밀리도록 보완

v1.5.7 수정: 공통 카드 묶음을 흐름도 격자 안의 기본 배치 줄로 이동

v1.5.8 수정: 기본 카드 배치를 직접 이동 시 최소 배치 단위와 같은 행/열 기준으로 재정렬

v1.5.9 수정: 카테고리 라벨을 상단 같은 높이로 정렬하고 카드가 카테고리 영역으로 이동하지 못하도록 제한

v1.6.0 수정: 모든 카테고리를 상단에 순서대로 배치하고 카테고리별 카드를 최대 두 줄 구조로 재배치

v1.6.1 수정: 카테고리별 항목을 두 개 세로 열 구조로 재배치

v1.6.2 수정: 카드 클릭 시 해당 카드 중심 이동, 연결 카드/화살표 강조, 재클릭 강조 해제 추가
